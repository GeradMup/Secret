# Love Pop Website

A tiny static romantic page for GitHub Pages.

## Files

- `index.html` - page markup
- `styles.css` - romantic styling and animations
- `script.js` - popup and floating hearts
- `assets/romantic-background.png` - generated romantic background image

## Publish on GitHub Pages

1. Create a new GitHub repository, for example `love-pop-website`.
2. Upload these files to the repository root.
3. Open the repository on GitHub.
4. Go to **Settings > Pages**.
5. Set **Source** to **Deploy from a branch**.
6. Select `main` and `/root`, then save.
7. Visit `https://your-username.github.io/love-pop-website/`.
